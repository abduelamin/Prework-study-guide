HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Head element must contain meta tags and title element e.g. viewport for resposive web design
  • ID and Classes on HTML are important for styling in CSS

CSS

  • CSS has selectors {property: value;} - the information in the curly braces is call a declration. You can have many decelrations in one curly braces but they must be separated by ;
  • 3 ways of adding styling; inline, internal and external
  • Elements incorporate the box model (margin, border, padding and content)
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Important to note is CSS, cascades styles. This means if you have 2 of same properties being styled but with different value's then the most recent value will override the previous one

Git

  • Create repo
  • clone repo into local computer via git clone followed by ssh url
  • verify connection via git remote -v where you should see 2 github URL's
  • Create github issues to layout project objective (user story + acceptance criteria using markdown language)
  • To edit the repo was must use feature branch to prevent changes to main branch. All our work must be done on feature branch.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • In the new branch make your changes, save them and stage them via 'git add -A' command. Next commit and push onto github and initiate PR
  • After PR the code will be reviewed + tested (QA testing). Once it passes it can be merged to main branch

JavaScript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • strings such as name's are wrapped with "". Whereas numbers and boolean values are not
    • Arrays are a type of variable that has a group of data - this data can be different types of data i.e. name, age, address
    • Arrays have a special length attribute that is selected via .length
    • 'for' loop must have 3 statements and each statement must be separated via a ;
    • To write a function you do the following
      1. function followed by name(){ //code block}
      2. Inside the code block you determine the code you want executed
      3. to call (execute) a function you simply just write function();
